Частые вопросы

Бот полностью бесплатный. Есть возможность поддержать разработку через добровольные пожертвования, но все основные функции доступны без оплаты.
Бот может:
Отвечать на вопросы по любым темам
Помогать с домашними заданиями
Генерировать изображения по описанию
Анализировать фотографии и отвечать на вопросы по ним
Переводить тексты
Работать с документами
Озвучивать текст голосом
Нет никаких лимитов
Просто напишите свой вопрос обычным текстом в чат с ботом. Можно также отправлять фотографии, документы и голосовые сообщения - бот умеет работать с разными форматами.
В меню бота (/menu) есть раздел "Настройки", где можно:
Выбрать предпочитаемую модель ИИ
Настроить стиль ответов
Включить/выключить дополнительные функции
Подпишитесь на официальный канал @neyronkabotnews, где публикуются все новости и обновления бота.
